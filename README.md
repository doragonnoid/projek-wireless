# Canteen Food Ordering App üçî

Aplikasi Android Native untuk pemesanan makanan di kantin universitas, dibuat untuk memenuhi Tugas Besar mata kuliah Wireless and Mobile Programming.

## Fitur Utama
1. **Autentikasi User**: Login & Register dengan validasi (SQLite Database).
2. **Browsing Menu**: Kategori makanan, minuman, dan promo.
3. **Keranjang Belanja**: Menambah/mengurangi jumlah item secara real-time.
4. **Checkout System**: Pilihan Dine-In (Pilih Meja) atau Delivery (Alamat Otomatis).
5. **Simulasi QRIS**: Generate QR Code dinamis berdasarkan total transaksi menggunakan ZXing.
6. **Payment Gateway Simulation**: Koneksi API menggunakan Retrofit (dengan fallback offline mode).

## Teknologi
- **Bahasa**: Kotlin
- **Database**: SQLite
- **API Client**: Retrofit
- **Libraries**: ZXing (QR Code), Gson, ViewBinding.
